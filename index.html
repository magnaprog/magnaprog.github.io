Abstract

This report presents the development and implementation of a Capture The Flag (CTF) variation of Pacman using both non-ML and ML approaches for agent behavior. The project involves the use of reinforcement learning (RL) algorithms such as Q-Learning and Proximal Policy Optimization (PPO) to enable adaptive agent strategies. This interdisciplinary approach integrates machine learning, game development, and animation concepts, making it highly relevant to the study of artificial life for computer graphics and vision.
